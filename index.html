Eplan图纸识图培训

第一部分：图纸特色功能解析

1.1 关联参考与跳转功能

图纸的核心优势：

  • 跨页跳转：图纸中的箭头和数字组合（如"2.0"）表示跳转到第2页第0列
  • 设备关联：同一设备的触点在不同页面间通过编号关联（如继电器线圈和响应触点）
  • 自动导航：通过点击或追踪可以快速找到相关的设备（设备标识符左/右）

1.2 中断点与连接功能

中断点概念：

  • 端子中断：通过端子排实现线路的连接和中断
  • 跨页连接：使用倒三角符号表示跨页连接

1.3 项目管理高效呈现

图纸项目组织：

  • 卷册管理：高层代号KZ代表图纸箱柜序号；位置代号BI代表安装信息页、CD代表电气回路页
  • 页面索引：清晰的页名和编号系统
  • 设备目录：完整的设备清单和参数

第二部分：图纸结构解析

2.1 图纸页面体系

BI页（主电路图）：

  • 柜正面/背面布置图：显示CPU、电源、I/O模块的现场实际1:1布局
  • 柜内系统拓扑图：显示控制柜内部的网络连接关系
  • 端子图表：简要列出柜内所有端子的连接关系，配合号码管方便核查（内/外侧概念）
  • 设备汇总表：清晰明了列出安装板上设备的标识符、部件分组、型号、数量、供应商表格

CD页（控制电路图）：

  • 电源进线图：220VAC和24VDC电源分配
  • 信号回路接线图：AI、AO、DI、DO回路的详细接线
  • 模块接线图：其他各种模块的电源或者信号连接
  • 通讯接线图：网络或者式串口通讯的连接方式

2.2 图纸符号体系

特色符号：

  • 设备标识：完整的设备型号和参数信息
  • 连接点编号：清晰的端子编号和连接关系
  • 信号流向：箭头指示信号传递方向
  • 中断标记：特殊符号表示线路的连接和中断
  • 线号标记：按照连接目标设备名+连接点和自身设备+连接点命名

第三部分：关键回路分析

3.1 电源回路追踪

220VAC电源路径：

  1. 主电源进线：UPS→TX-P1（端子排）→QF1
  2. 备用电源进线：市电→TX-P1（端子排）→QF2
  3. 电源分配：通过断路器QF3-QF9分配到各设备

24VDC电源路径：

  1. 电源转换：220VAC→DS1/DS2→DSRY1→24VDC
  2. 电源冗余：双电源输入，故障时自动切换
  3. 电源分配：通过TX-P2、TX-P3分配到各模块
  4. 中断点管理：通过端子排实现电源的去向标识和跳转

3.2 常用信号回路追踪

模拟量输入回路：传感器→SPD防雷→GS安全栅（若有）→PLC-AI
数字量输入回路：传感器→SPD防雷→PLC-DI
数字量输出回路：PLC-DO→继电器线圈→响应触点→执行元件
串口通讯回路：通讯设备→防雷→GS安全栅（若有）→采集模块

第四部分：故障定位方法

4.1 常见故障点识别

电源故障：

  • 主电源故障：检查UPS和市电状态，利用电源分配图定位
  • 24VDC故障：检查开关电源和冗余切换，利用端子图表定位
  • 电源分配故障：检查各断路器状态，利用电源进线图定位

通讯故障：

  • 网络故障：检查交换机连接，利用柜内系统拓扑图定位
  • RS485故障：检查通讯模块和设备连接，利用通讯接线图定位
  • PLC模块故障：检查PLC模块状态，利用模块接线图定位

I/O故障：

  • AI故障：检查传感器信号和安全栅，利用AI回路图定位
  • DI故障：检查输入信号和隔离模块，利用DI回路图定位
  • DO故障：检查输出信号和继电器，利用DO回路图定位

4.2基于图纸特性的故障诊断方法

快速定位流程：

  1. 模块定位：通安装信息页快速确定故障设备的物理位置
  2. 跨页跳转：利用数字组合功能直接跳转到故障设备的详细接线位置
  3. 设备关联：找到同一设备的所有相关连接，了解完整工作状态
  4. 信号追踪：从故障点反向追踪信号来源，分析信号传递路径和连接关系
  5. 回路分析：通过回路图分析故障原因，定位信号中断或异常位置

核心优势：无需手动翻页查找，通过图纸的跨页跳转和关联功能直接定位到关键位置，大幅提高故障排查效率。从设备定位到信号分析形成完整诊断链条。

总结与问答

核心要点回顾：

  1. 图纸特色功能：关联参考、跳转功能、中断点管理
  2. 图纸结构：BI页安装信息、CD页控制接线图，清晰的页面标识
  3. 回路分析：从电源到执行元件的完整信号链，利用中断点管理
  4. 故障定位：利用图纸的关联功能和中断点快速定位故障

实用技巧：

  • 善用跳转：通过数字组合快速找到相关页面
  • 追踪信号：从故障点反向追踪信号来源
  • 重视中断点：检查电源、信号、通讯的中断点状态
  • 利用关联：找到同一设备的所有相关连接
